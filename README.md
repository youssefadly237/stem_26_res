# S'26 STEM Results

This project provides an improved interface for viewing results.

The data was obtained from a publicly accessible source that required no
authentication or special access permissions at the time of collection.
This project is unofficial and is not affiliated with any STEM school,
educational institution, or government organization.

If an authorized owner or administrator of the original data source requests
removal of the data or repository, I will review the request and may take it
down.
